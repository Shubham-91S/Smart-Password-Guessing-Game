# 🔐 Smart Password Guessing Game

A web-based password guessing game built using **Python** and **Flask**. The game offers multiple categories, difficulty levels, a hint system, score calculation, leaderboard, and game history to make learning and playing more engaging.

## 📌 Project Overview

The Smart Password Guessing Game is a fun and interactive web application where players try to guess hidden words within a limited number of attempts.

The game provides different categories and difficulty levels, making it enjoyable for users of all ages. It also stores player scores and game history using file handling, allowing users to track their performance.

This project was developed to strengthen my Python programming skills and gain practical experience with Flask web development.
